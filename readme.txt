Comment obtenir la configuration ?
Téléchargez le fichier zip.
Allez dans le répertoire d'installation de MongoDB et sous le dossier bin, exécutez cette commande : mongod    Cela démarrera le serveur MongoDB. Laissez cette instance CLI ouverte et démarrez une autre instance CLI.
Dans la nouvelle CLI, accédez à l'endroit où vous avez extrait le projet, ex. C:/nodejs-mongodb-default-template puis tapez : npm install puis attendez qu'il termine l'installation de tous les modules requis pour exécuter notre application Web Node.js.
Une fois l'installation terminée, tapez la commande suivante pour exécuter notre application Web :  npm start   Assurez-vous de garder la CLI ouverte.
Accédez ensuite à http://127.0.0.1:3100/ à l'aide de votre navigateur préféré.
